# Media Website

A simple static website project with a home page, products page, and about page.

## Structure

- `index.html` - Home page
- `pages/products.html` - Products page
- `pages/about.html` - About page
- `css/styles.css` - Stylesheet
- `images/` - Image assets

## How to use

1. Open `index.html` in your web browser.
2. Navigate to the other pages via the site links.

## Notes

- This is a static HTML/CSS site; no build step is required.
